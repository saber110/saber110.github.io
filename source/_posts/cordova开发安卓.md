## 命令
```bash
cordova create hello com.example.hello HelloWorld
cordova platform add android/ios/browser --save
cordova build <ios/android>
cordova run <ios/android>
cordova plugin add cordova-plugin-camera
cordova plugin add cordova-plugin-camera
```
